# Baselines

这一层用于承接 `Sayachan` 当前系统真相，以及 PMO 需要依赖的稳定事实基线。

这里的文件主要回答：

- 系统现在实际由什么组成
- 系统现在是怎么跑起来的
- 后端 API 和模型当前长什么样
- 当前已交付范围、债项和近期清理重点是什么

当前基线包括：

- `system-baseline.md`
- `runtime-baseline.md`
- `backend-api.md`
- `private-core-boundary.md`
- `roadmap.md`

这一层帮助 PMO 做判断，但不直接驱动 PMO 状态机。

如果某份文档主要在定义流程、交接规则或横向约束，它通常不应放在这里。
