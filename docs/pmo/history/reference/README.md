# History Reference

这一层用于承载不属于 candidate 归档、也不属于 execution report 归档的历史参考材料。

它主要包括：

- 迁移记录
- 过渡期说明

- 它们属于历史背景或流程反思
- 不是当前 PMO 的 active contract
- 也不是某次 sprint closeout 的直接 execution report
