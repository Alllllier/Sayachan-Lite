# History

这一层用于承载已经退出 active PMO surface 的历史材料。

这里的内容主要分成 3 类：

- `candidates/`
  - 承接已经退出 `state/sprint_candidates.md` 的 completed / replaced candidate 记录
- `reports/`
  - 承接已经退出 `state/execution_report.md` active surface 的已读 execution report
- `reference/`
  - 承接不属于 candidate 或 execution report 的历史参考材料
  - 例如迁移记录、过渡说明、重建经验、audit summary、流程评估

这一层不应该再承担当前 PMO 的 active contract。

使用上应优先按子层理解，而不是把 `history/` 根目录继续当作随手堆放历史文件的位置。
